# Frontend Odyssey — The Life of a Developer

> *A cinematic, interactive storytelling website built entirely with HTML, CSS, and JavaScript.*

## 🚀 Overview

**Frontend Odyssey** is an award-style single-page experience that takes you through the emotional journey of becoming a developer — from wide-eyed beginner to caffeine-powered, deadline-surviving legend.

No external images. No frameworks. No backend. Pure code.
